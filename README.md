# Task 16: Time Series Forecasting
Dataset: Store Item Demand Forecasting (train.csv)

Steps Performed:
- Loaded dataset and converted date column to datetime
- Aggregated sales by month
- Plotted sales trend and rolling mean
- Split data into train/test (80/20 time-based)
- Applied Exponential Smoothing model
- Forecasted future sales
- Evaluated model using MAE and MAPE
- Exported forecast results to CSV
- Generated forecast report

Files:
- task16_forecasting.ipynb
- forecast_output.csv
- forecast_report.txt
